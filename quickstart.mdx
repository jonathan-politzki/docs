## REST API ENDPOINTS

Base URL: https://api.jeanmemory.com

### Add Memory
POST /memories
Headers: { "Authorization": "Bearer jean_sk_...", "Content-Type": "application/json" }
Body: { "messages": [{ "role": "user", "content": "Remember this" }] }

### Search Memory
GET /memories/search?query=favorite%20color
Headers: { "Authorization": "Bearer jean_sk_..." }

### Delete Memory
DELETE /memories/{memory_id}
Headers: { "Authorization": "Bearer jean_sk_..." }
